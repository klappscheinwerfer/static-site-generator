# SSG

A static site generator for GitHub, written in C++

## Installation

* Clone this repository `git clone https://github.com/klappscheinwerfer/Static-Site-Generator`
* Build
	* `mkdir build`
	* `cd build`
	* `cmake ..`
	* `make`
